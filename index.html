Kalender

~ simple android calendar app ~

Alpha Release — April 27, 2026

// Overview

Kalender is a free, open-source Android app that displays Indonesian national holidays in a clean, Material 3-designed calendar. Currently in alpha, the app is lightweight, privacy-focused, and optimized for performance.

// Features

• Monthly calendar with swipe navigation
• List of national, religious, and joint holidays
• Material 3 design with dynamic color theming
• Supports Android 12+ dynamic color (Monet)
• Lightweight and optimized for performance

// Tech Stack

• Language: Kotlin
• UI: XML Layouts
• Design: Material 3 with Dynamic Colors
• Navigation: ViewPager2
• Architecture: Native Android

// Current Limitations

• Only available in Indonesian
• Holiday data only for the year 2026
• Alpha release — features and design may change

// Getting Started

Requirements: Android Studio (latest stable), Android device or emulator (API 26+)

Installation:
1. Clone the repository
2. Open in Android Studio
3. Build and run
