Privacy Policy

~ Kalender App ~

Last updated: April 27, 2026

// Overview

Kalender is a free and open source calendar app built for Android. This privacy policy explains how the app handles your data. The short version: it doesn't collect any.

// Data Collection

Kalender does not collect, store, transmit, or share any personal data. The app has no analytics, no tracking, no ads, and no remote servers of any kind.

// Data Stored on Device

The app stores the following data locally on your device only, using Android SharedPreferences:

• Selected country — your preferred holiday region (e.g. Indonesia)
• Selected language — your preferred app language (English or Indonesian)
• Widget offset — the current month position of the home screen widget

This data never leaves your device and is only used to restore your preferences between sessions.

// Internet Access

Kalender does not use the internet. No network requests are made at any time. All holiday data is bundled directly inside the app.

// Permissions

Kalender does not request any sensitive Android permissions. The only system integration used is the home screen widget, which is entirely optional.

// Third Party Services

Kalender uses no third party SDKs, analytics libraries, advertising networks, or crash reporting services.

// Children's Privacy

Kalender does not collect any data from anyone, including children. The app is safe for all ages.

// Changes to This Policy

If this privacy policy changes, the updated version will be published here. Since the app collects no data, changes are unlikely to affect users in any meaningful way.
